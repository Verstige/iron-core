# Iron Core Fitness Redesign Concept

Static high-fidelity website concept for Iron Core Fitness.

## Run locally

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Included

- Full homepage rebuild using the approved brand prompt
- Responsive mobile-first layout
- Scroll reveal animations and reading progress rail
- Sticky mobile shop CTA
- Workout filters
- Amazon country selector using the current Iron Core marketplace links
- SEO metadata and Organization schema

## Files

- `index.html`
- `styles.css`
- `script.js`
